Custom Network Time Protocol

Developed a Custom Network Time Protocol(CNTP) in C which enable synchronization of time between nodes over a network. We follow a client-server model in which Server broadcasts its time and the Clients sync their time with that of Server. CNTP offers improvement over NTP(Network Time Protocol)v4 by calculating end-to-end delay for the links between the nodes rather than assuming that outgoing and incoming paths between the nodes are symmetric. Timestamping of packets is done at the Kernel Driver level.
